## Component Details

### 1. User Interface Layer
- **React Component**: Production-ready component for integration
- **HTML Test Page**: Standalone page for quick testing
- **Streaming Support**: Real-time progress display

### 2. Flask Web Server
- **File Upload Handling**: Secure filename, validation
- **Background Processing**: Non-blocking training via threads
- **Progress Streaming**: Queue-based SSE implementation
- **CORS Enabled**: Frontend integration ready

### 3. Model Module
- **Dual Retraining Functions**: 
  - `retrain_model()` - Synchronous
  - `retrain_model_with_queue()` - Async with progress
- **Smart Caching**: GLOBAL_MODEL for fast inference
- **Hybrid Forecasting**: Prophet + LSTM ensemble
- **Full Persistence**: Save/load all artifacts

### 4. Persistence Layer
- **Data Storage**: Combined dataset with deduplication
- **Model Artifacts**: Prophet pickle + LSTM H5 + scaler
- **Metadata Tracking**: Training info, row counts
- **Upload Management**: Temporary storage for uploads
